команды:
cd (change directory) - смена рабочей папки
git clone https://ссылка на репозиторий - скопировать проэкт на локаль
git status - проверка изменений в локальном проэкте
git add file.name - добавить файл в cmd для работы (чтобы он его видел) + сохранение изменений в файле
git commit -m "добавил изменения" - добавление комментария, что изменили в коде файла
git push - отправить изменения в удалённый репозиторий github
